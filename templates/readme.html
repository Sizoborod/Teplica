Отправка данных на сервер

Для отправки данных в таблицу нужно сделлать GET запрос на сервер с передачей параметров.

Пример запроса через строку.

https://nikitinan.pythonanywhere.com/add_sensors?token=уникальный_токен&t_in=22&h_in=77&t_out=22&h_out=77&mois1=555&mois2=555&light=888&pump=0&heat=0&led=1&fan=0

Расшифровка параметров

  • token - уникальный токен для пользоваттеля
  • t_in - температура внутри теплицы (0-50)
  • h_in - влажность воздуха внутри теплицы (0-100)
  • t_out - температура снаружи теплицы (0-50)
  • h_out - влажность воздуха снаружи теплицы (0-100)
  • mois1 - влажность почвы первого датчика теплицы (0-1023)
  • mois2 - влажность почвы второго датчика теплицы (0-1023)
  • light - освещеность снаружи теплицы (0-1023)
  • pump - индикатор работы водяного насоса
  • heat - индикатор работы подогрева теплицы
  • led - индикатор работы подсветки теплицы
  • fan - индикатор работы охлаждения теплицы

Сервер возвращает "OK" в случае успешного запроса и "NO" в других случаях

Пример запроса через строку.

https://nikitinan.pythonanywhere.com/update/token
